Moves the pivot point of a GameObject to a new position provided. The method also preserves the orientations of the children objects. Any errors are reported as exceptions.

Parameters
forObjectThe objects whose pivot point will be modified. If the provided object is meshless then this method simply changes the position of the GameObject while preserving the transform states of the children objects.
moveToThe new point in space to move the pivot to.
recalculateNormalsTangentsShould the mesh normals and tangents be recalculated after pivot modification.

bool  recalculateNormalsTangents

static

Move the pivot point to the average of the positions of the children of the provided GameObject. If the provided object has no children then the method does nothing.

Parameters
forObjectThe GameObject whose pivot will be moved.
takeNestedAverageIf this argument is passed true then the pivot is moved to the average of the world positions of all the children of the given GameObject including the deep nested ones. If this is false then the new pivot position is the average of the first level children positions.
includeInactiveWhether to consider the inactive children when calculating the average for the new pivot position.
recalculateNormalsTangentsShould the mesh normals and tangents be recalculated after pivot modification.

Changes the rotation of the pivot point of a GameObject. The method also preserves the orientations of the children objects. Any errors are reported as exceptions. Please note that this operation can result in skewing of the object if it is non uniformly scaled.

Parameters
forObjectThe objects whose pivot point will be modified. If the provided object is meshless then this method simply changes the rotation of the GameObject.
newRotationThe new rotation of the pivot point in world space.
recalculateNormalsTangentsShould the mesh normals and tangents be recalculated after pivot modification.
